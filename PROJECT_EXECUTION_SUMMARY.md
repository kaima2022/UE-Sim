## 🏗️ 技术架构实现

### 原型系统架构 (UET项目)
```
Soft-UE原型系统 (149个源文件)
├── SES层 (语义层)
├── PDS层 (包分发层)
├── PDC层 (传输上下文层)
└── 基础网络接口
```

### ns-3迁移实现 (Soft-UE-ns3项目)
```
ns-3集成实现 (37个核心源文件)
├── src/soft-ue/model/ses/           # SES层实现
│   ├── ses-manager.h/cc             # 核心管理器
│   ├── operation-metadata.h         # 元数据管理
│   └── msn-entry.h                  # MSN表管理
├── src/soft-ue/model/pds/           # PDS层实现
│   ├── pds-manager.h/cc             # 分发管理器
│   ├── pds-header.cc                # 协议头部
│   └── pds-statistics.cc            # 性能统计
├── src/soft-ue/model/pdc/           # PDC层实现
│   ├── pdc-base.h/cc                # 基础类
│   ├── ipdc.h/cc                    # 不可靠PDC
│   ├── tpdc.h/cc                    # 可靠PDC
│   └── rto-timer/                   # 重传计时器
├── src/soft-ue/model/network/       # 网络设备层
│   ├── soft-ue-net-device.h/cc      # 网络设备
│   └── soft-ue-channel.h/cc         # 通信通道
├── src/soft-ue/helper/              # Helper层
│   └── soft-ue-helper.h/cc          # 用户接口
└── src/soft-ue/test/                # 测试框架
    ├── ses-manager-test.cc          # SES测试
    ├── pds-manager-test.cc          # PDS测试
    ├── ipdc-test.cc                 # IPDC测试
    ├── tpdc-test.cc                 # TPDC测试
    └── soft-ue-integration-test.cc  # 集成测试
```
