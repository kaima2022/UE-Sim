# Soft-UE ns-3 项目总览图解

## 项目本质

**一句话概括**: 将原型系统 `/home/makai/Soft-UE` 中的 Ultra Ethernet 协议栈完整迁移到 ns-3 离散事件仿真器中，使其能够进行大规模网络性能仿真。

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          项目迁移关系                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   /home/makai/Soft-UE (原型)          /home/makai/Soft-UE-ns3 (仿真)    │
│   ┌─────────────────────┐             ┌─────────────────────┐          │
│   │  Ultra Ethernet     │   迁移到    │  Ultra Ethernet     │          │
│   │  协议栈实现          │ ─────────▶ │  + ns-3 集成        │          │
│   │  (独立运行)          │             │  (仿真环境)          │          │
│   └─────────────────────┘             └─────────────────────┘          │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 关键概念解释

| 概念 | 全称 | 作用 |
|------|------|------|
| **SES** | Semantic Sub-layer | 语义子层，处理操作类型、授权验证 |
| **PDS** | Packet Delivery Sub-layer | 包分发子层，负责路由和统计 |
| **PDC** | Packet Delivery Context | 包分发上下文，管理单次传输的状态 |
| **IPDC** | Immediate PDC | 不可靠传输，无确认机制，低延迟 |
| **TPDC** | Transactional PDC | 可靠传输，有确认和重传机制 |
| **FEP** | Fabric Endpoint | 网络端点标识符 |
| **MSN** | Message Sequence Number | 消息序列号，用于乱序重组 |
| **RTO** | Retransmission Timeout | 重传超时计时器 |
